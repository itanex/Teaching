# Exercise 2

* Create a new repository on GitHub called `'exercise-2'`
   * Go to GitHub and use the Create New Repository button
* Clone Repository to your local Working Directory
   * Use `"git remote origin <repository.git>"`
* Create a file called `myFavoriteColor.txt` that contains the name of your favorite color
* Stage and Commit
   * `git add .`
   * `git commit -m "<message>"`
* Push Changes
   * `git push -u origin master`
* Change your favorite color to something else
* Stage and Commit
   * `git add .`
   * `git commit -m "<message>"`
* Push Changes
   * `git push`
