# End of Line Punctuation

#### JAVASCRIPT STATEMENTS AND EXPRESSIONS
End with a Semi-Colon
```JavaScript
a = b + c;

var x = 0;
```

#### FUNCTION DECLARATION, LOOP BLOCK, CONTROL STRUCTURES
End in Closing Curly Brace '}'
```JavaScript
function fn(a, b, c) {
  // does something
}

for(...) {
  // does something
}

while (condition) {
  // does something
}

if (condition) {
	// does something
}

switch (condition) {
  // does something
}
```

#### PARAMETER/ARGUMENT LISTS
Separated by Commas
```JavaScript
function fn(a, b, c) {
  // does something
}

fn(1, 2, 4);
```

#### MULTI VARIABLE DECLARATIONS
Separated by Commas
Ends with Semi-Colon

```JavaScript
var x, a, r;
```

#### OBJECT LITERAL NOTATION (JSON)
Fields Separated by Commas
Ends with Semi-Colon
```JavaScript
var point = {
	x: 0,
	y: 0,
	z: 0
};
```
